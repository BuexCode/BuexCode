# BuexCode

The name BuexCode is derived from the geman word "ausbüxen" /aúsbüxen/ `<ausb&uumlxen>`  and is a colloquial synonym for
- german -> abhauen, sich davonmachenm, entkommen
- english -> escape
- dutch -> ontsnappen
